### 第一步：确保服务运行

#### 1. 检查后端是否正在运行
打开浏览器访问: http://localhost:8080/docs

#### 2. 检查前端是否正在运行
打开浏览器访问: http://localhost:3000

如果没有运行前端，打开一个新的终端窗口，运行：
```bash

npm run dev
```

### 第二步：获取抖音 Cookie (自动方式，推荐！)

#### 1. 打开前端网站
访问 http://localhost:3000

#### 2. 进入 Cookie 管理页面
点击顶部菜单的 **"Cookie 管理"**

#### 3. 点击"添加账号"
点击 **"添加账号"** 按钮，弹出添加账号窗口

#### 4. 点击"打开浏览器登录"
在弹出的窗口中，找到醒目的紫色区域：
- **自动获取 Cookie（推荐）**
- 点击 **"打开浏览器登录"** 按钮

#### 5. 在浏览器中登录抖音
系统会自动打开 Chrome 浏览器并跳转到抖音登录页面

推荐使用 **手机扫码登录**（更安全）

#### 6. 等待自动获取
登录成功后：
- ✅ 浏览器会自动关闭
- ✅ Cookie 会自动填充到表单中
- ✅ 弹出提示框告诉你 Cookie 已获取

#### 7. 保存账号
- 输入账号名称（例如：`我的抖音`）
- 点击 **"添加账号"** 完成

## ⚠️ 注意事项

### Cookie 安全
- ⚠️ Cookie 包含账号信息，请妥善保管
- ⚠️ 不要在公共设备上使用
- ⚠️ Cookie 会过期，需要定期更新

### 请求频率
- ⚠️ 请设置合理的请求间隔
- ⚠️ 频繁请求可能导致账号受限
- ⚠️ 建议每次请求间隔 2-5 秒

### 浏览器依赖
- ⚠️ 需要安装 Chrome 浏览器
- ⚠️ 需要安装 Python 依赖：`undetected_chromedriver`, `selenium`

### 合规使用
- ⚠️ 仅供学习和研究使用
- ⚠️ 请遵守平台服务条款
- ⚠️ 尊重用户隐私

## 📞 常见问题

### Q1: 点击"打开浏览器登录"没有反应？
**检查：**
- 后端是否在运行？访问 http://localhost:8080/docs
- Chrome 浏览器是否已安装？
- Python 的 selenium 和 undetected_chromedriver 是否已安装？

**解决：**
```bash
pip install selenium undetected-chromedriver
```

### Q2: Cookie 获取失败？
**可能原因：**
- 登录超时
- 浏览器被阻止

**解决：**
- 尝试手动获取 Cookie
- 检查网络连接

### Q3: 如何手动获取 Cookie？
**步骤：**
1. 浏览器打开 https://www.douyin.com 并登录
2. 按 F12 打开开发者工具
3. 点击 Network 标签
4. 进入任意视频详情页
5. 找到 `aweme/detail` 请求
6. 复制 Request Headers 中的 Cookie
7. 在前端"添加账号"窗口中粘贴

### Q4: Cookie 过期了怎么办？
**解决：**
1. 删除过期的账号
2. 重新获取 Cookie（自动或手动）
3. 添加新账号

### Q5: 前端显示"启动失败，请检查后端服务是否运行"？
**解决：**
1. 确认后端正在运行
2. 刷新浏览器页面
3. 检查 CORS 是否正常（我们已配置）
