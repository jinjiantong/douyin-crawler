## 🚀 如何启动

### 确保后端运行

首先，确保 Crawler 后端服务正在运行：

```bash
cd Crawler-main
.\venv\Scripts\python.exe main.py
```

后端地址：`http://localhost:8080`

### 启动前端网站

有三种方式：

#### 方式一：使用启动脚本（推荐）
```
双击文件：启动前端.bat
```

#### 方式二：手动启动
```bash
cd douyin-crawler-frontend
npm install      # 如果未安装依赖
npm run dev      # 启动开发服务器
```

#### 方式三：构建生产版本
```bash
cd douyin-crawler-frontend
npm run build    # 构建
npm run preview  # 预览
```

### 访问网站

打开浏览器访问：**http://localhost:3000**

## 📖 使用流程

### 第一步：添加 Cookie

1. 打开网站首页
2. 点击顶部导航的"账号管理"
3. 点击"添加账号"按钮
4. 输入账号名称（如：我的抖音）
5. 粘贴从抖音网页版复制的 Cookie
6. 点击"添加账号"

**如何获取 Cookie？**
1. 打开抖音网页版并登录
2. 按 F12 打开开发者工具
3. 点击 Network 标签
4. 进入任意视频详情页
5. 找到 `aweme/detail` 请求
6. 复制 Request Headers 中的 Cookie

### 第二步：搜索视频

1. 点击顶部导航的"搜索视频"
2. 在搜索框输入关键词
3. 点击搜索按钮
4. 浏览搜索结果（瀑布流展示）
5. 点击视频卡片查看详情

### 第三步：采集评论

1. 在视频详情页点击"查看评论"
2. 点击"开始采集评论"按钮
3. 系统自动加载评论数据
4. 点击"加载更多"获取更多评论
5. 选择"导出JSON"或"导出CSV"

### 第四步：查看用户信息

1. 在视频详情页点击作者头像或名称
2. 查看用户详细信息
3. 浏览用户作品列表
4. 点击作品继续查看详情

### 数据流
```
用户操作 → React组件 → Axios请求 → Crawler API

响应数据 → React状态 → UI更新
```

## 📊 API 对应关系

| 前端功能 | 调用 API | 说明 |
|---------|---------|------|
| 添加账号 | POST /douyin/add_account | 添加 Cookie |
| 账号列表 | GET /douyin/account_list | 获取所有账号 |
| 过期账号 | POST /douyin/expire_account | 标记过期 |
| 搜索视频 | GET /douyin/search | 关键词搜索 |
| 视频详情 | GET /douyin/detail | 获取详情 |
| 获取评论 | GET /douyin/comments | 一级评论 |
| 评论回复 | GET /douyin/replys | 二级回复 |
| 用户信息 | GET /douyin/user | 用户+作品 |

## 🆘 常见问题

### Q1: 前端无法访问？
**检查：**
- npm 依赖是否安装成功
- 端口 3000 是否被占用
- Node.js 是否安装

### Q2: 后端无法连接？
**检查：**
- Crawler-main 后端是否运行
- 后端端口是否为 8080
- 防火墙是否阻止

### Q3: 搜索没有结果？
**原因：**
- Cookie 可能已过期
- 关键词无相关视频
- 网络请求被限制

### Q4: 评论采集失败？
**原因：**
- Cookie 权限不足
- 视频评论已关闭
- 请求频率过高
