## 概述

本系统是一个抖音数据采集工具，支持视频搜索、用户作品列表查看、评论采集和数据导出等功能。

---

## 功能特性

| 功能模块 | 功能描述 | 状态 |
|---------|---------|------|
| 账号管理 | 添加/删除抖音账号Cookie | ✅ |
| 视频搜索 | 关键词搜索抖音视频 | ✅ |
| 用户详情 | 查看博主主页信息和作品列表 | ✅ |
| 作品列表 | 默认加载所有作品，自动分页 | ✅ |
| 评论采集 | 单视频评论采集 | ✅ |
| 多视频采集 | 批量采集多个视频评论 | ✅ |
| 数据导出 | 支持JSON/CSV格式导出 | ✅ |
| 文件夹选择 | 使用浏览器API选择保存位置 | ✅ |

## 核心功能详解

### 1. 账号管理

- **功能描述**: 添加抖音账号Cookie用于数据采集
- **文件**: [CookieManager.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/CookieManager.jsx)
- **特性**: 支持多个账号轮换使用

### 2. 视频搜索

- **功能描述**: 通过关键词搜索抖音视频
- **文件**: [VideoSearch.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/VideoSearch.jsx)

### 3. 用户详情页

- **功能描述**: 查看博主主页信息和所有作品列表
- **文件**: [UserInfo.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/UserInfo.jsx)
- **特性**:
  - 默认自动加载所有作品
  - 实时显示加载进度
  - 支持全选/反选视频
  - 支持跳转到多视频采集

### 4. 单视频评论采集

- **功能描述**: 采集单个视频的评论
- **文件**: [CommentSection.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/CommentSection.jsx)
- **特性**: 支持全量采集，自动分页

### 5. 多视频评论采集

- **功能描述**: 批量采集多个视频的评论，支持暂停/继续
- **文件**: [MultiVideoCollector.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/MultiVideoCollector.jsx)
- **特性**:
  - 实时显示采集进度
  - 支持暂停/继续操作
  - 支持全量采集评论
  - 支持导出JSON/CSV格式

### 6. 数据导出功能

- **功能描述**: 将采集的数据导出为文件
- **支持格式**: JSON、CSV
- **特性**:
  - 自定义文件名前缀
  - 支持选择保存文件夹
  - 使用浏览器原生文件系统API

## API 接口

### 抖音接口列表

| 接口 | 方法 | 描述 | 参数 |
|------|------|------|------|
| `/douyin/user` | GET | 获取用户信息和作品 | id, offset, limit |
| `/douyin/comments` | GET | 获取视频评论 | id, offset, limit |
| `/douyin/search` | GET | 搜索视频 | keyword, offset, limit |
| `/douyin/detail` | GET | 获取视频详情 | id |
| `/douyin/replys` | GET | 获取评论回复 | id, comment_id |
| `/douyin/add_account` | POST | 添加账号 | id, cookie |
| `/douyin/account_list` | GET | 获取账号列表 | - |
| `/douyin/expire_account` | POST | 过期账号 | id |

### 评论接口响应结构

```json
{
  "code": 0,
  "msg": "成功",
  "data": {
    "total": 2605,
    "comments": [...],
    "has_more": 1
  }
}
```

- `has_more`: 1表示还有更多评论，0表示已全部加载

## 使用说明

### 启动方式

**方法一：使用启动脚本（推荐）**

```bash
# 启动后端服务
cd Crawler-main
.\start_server.bat

# 启动前端服务
cd douyin-crawler-frontend
.\start_frontend.bat
```

**方法二：手动启动**

```bash
# 后端
cd Crawler-main
.\venv\Scripts\python.exe main.py

# 前端
cd douyin-crawler-frontend
npm run dev
```

### 访问地址

- **前端**: http://localhost:3000
- **后端**: http://localhost:8080

### 使用流程

1. **添加账号** → 在账号管理页面添加抖音Cookie
2. **搜索视频** → 使用关键词搜索或输入用户ID
3. **查看用户详情** → 点击用户头像进入详情页
4. **选择视频** → 在作品列表中选择要采集的视频
5. **批量采集** → 点击"批量采集"按钮开始采集
6. **导出数据** → 采集完成后导出JSON/CSV

## 注意事项

1. **Cookie有效期**: 抖音Cookie通常有效期为几天到几周，失效后需要重新获取
2. **请求频率**: 建议适当控制请求频率，避免被限流
3. **浏览器兼容性**: 文件夹选择功能需要HTTPS或localhost环境
4. **数据量**: 大量评论采集可能需要较长时间，请耐心等待
