## 3. 核心功能实现

### 3.1 多视频评论采集

#### 3.1.1 功能描述

批量选择多个视频，同时采集每个视频的评论，支持暂停/继续操作。

#### 3.1.2 实现方案

**文件**: [MultiVideoCollector.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/MultiVideoCollector.jsx)

**核心算法**:

```javascript
const collectAllComments = async () => {
  setIsCollecting(true);
  
  // 顺序处理每个视频
  for (let i = 0; i < videos.length; i++) {
    // 检查暂停状态
    while (isPausedRef.current) {
      await new Promise(resolve => setTimeout(resolve, 100));
    }
    
    const video = videos[i];
    setCurrentVideoIndex(i);
    await collectVideoComments(video);
  }
};

const collectVideoComments = async (video) => {
  let allCommentsList = [];
  let currentOffset = 0;
  let hasMore = 1;
  let emptyCount = 0;
  
  // 使用 has_more 字段控制循环
  while (hasMore === 1) {
    // 检查暂停状态
    while (isPausedRef.current) {
      await new Promise(resolve => setTimeout(resolve, 100));
    }
    
    const response = await axios.get(`${API_BASE}/douyin/comments`, {
      params: { id: videoId, offset: currentOffset, limit: 50 }
    });
    
    const data = response.data.data;
    const newComments = data.comments || [];
    hasMore = data.has_more;
    
    allCommentsList = [...allCommentsList, ...newComments];
    currentOffset += 50;
    
    // 更新进度
    setCollectionStatus(prev => ({
      ...prev,
      [videoId]: { 
        progress: (allCommentsList.length / data.total) * 100,
        collected: allCommentsList.length,
        total: data.total
      }
    }));
    
    // 防止请求过快
    await new Promise(resolve => setTimeout(resolve, 500));
  }
};
```

#### 3.1.3 关键技术点

| 技术点 | 说明 |
|--------|------|
| `isPausedRef` | 使用 ref 存储暂停状态，避免闭包问题 |
| `has_more` | 使用后端返回的字段判断是否继续 |
| `emptyCount` | 连续3次空响应则停止，防止死循环 |
| `setTimeout` | 请求间隔500ms，防止被限流 |

#### 3.2.1 功能描述

#### 3.2.2 实现方案

```javascript
const fetchAllVideos = async () => {
  setLoadingVideos(true);
  let allVideos = [];
  let currentOffset = 0;
  let hasMore = true;

  try {
    // while 循环顺序加载
    while (hasMore) {
      const response = await axios.get(`${API_BASE}/douyin/user`, {
        params: { id: userId, offset: currentOffset, limit: 30 }
      });

      const newVideos = response.data.data.aweme_list || [];
      allVideos = [...allVideos, ...newVideos];
      setVideos([...allVideos]); // 实时更新UI
      
      hasMore = newVideos.length === 30;
      currentOffset += 30;
      
      console.log(`已加载 ${allVideos.length} 个作品`);
    }
  } finally {
    setLoadingVideos(false);
  }
};
```

### 3.3 数据导出功能

#### 3.3.1 功能描述

支持JSON和CSV格式导出，可选择保存文件夹。

#### 3.3.2 实现方案

**文件**: [UserInfo.jsx](file:///f:/金博士AI/评论分析/douyin-crawler-frontend/src/components/UserInfo.jsx)

**导出流程**:

```javascript
const exportData = async (format) => {
  const exportData = {
    user_id: userId,
    user: user,
    videos: videos,
    export_time: new Date().toISOString(),
    total_videos: videos.length
  };

  let content, filename;
  
  // 1. 生成内容
  if (format === 'json') {
    content = JSON.stringify(exportData, null, 2);
    filename = `${exportFileName}.json`;
  } else {
    content = convertToCSV(exportData);
    filename = `${exportFileName}.csv`;
  }

  // 2. 根据用户选择决定导出方式
  if (customSavePath && selectedDirectory) {
    // 使用 File System Access API
    const fileHandle = await selectedDirectory.getFileHandle(filename, { create: true });
    const writable = await fileHandle.createWritable();
    await writable.write(content);
    await writable.close();
  } else {
    // 使用传统下载方式
    const blob = new Blob([content], { type: 'text/plain;charset=utf-8' });
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.href = url;
    link.download = filename;
    link.click();
    URL.revokeObjectURL(url);
  }
};

// CSV 转换函数
const convertToCSV = (data) => {
  const headers = [
    '视频ID', '视频描述', '点赞数', '评论数', '分享数', '收藏数', '时长(秒)', '发布时间'
  ];
  
  const rows = data.videos.map(video => [
    video.aweme_id,
    `"${video.desc.replace(/"/g, '""')}"`,
    video.statistics?.digg_count || 0,
    video.statistics?.comment_count || 0,
    video.statistics?.share_count || 0,
    video.statistics?.collect_count || 0,
    Math.floor(video.video?.duration / 1000),
    new Date(video.create_time * 1000).toLocaleString('zh-CN')
  ]);

  return [headers.join(','), ...rows.map(row => row.join(','))].join('\n');
};
```

```python
import httpx

async def common_request(url, params, headers):
    """通用异步HTTP请求"""
    async with httpx.AsyncClient(timeout=30.0) as client:
        try:
            response = await client.get(url, params=params, headers=headers)
            return response.json(), True
        except Exception as e:
            return {"error": str(e)}, False
```

## 9. 性能优化方案

### 9.1 前端优化

| 优化项 | 方案 | 效果 |
|--------|------|------|
| 分页加载 | 边加载边显示作品列表 | 快速响应用户 |
| 虚拟列表 | 大数据量时使用虚拟滚动 | 减少DOM节点 |
| 请求间隔 | 500ms延迟防止限流 | 提高成功率 |
| 错误重试 | 最多重试3次 | 提高容错性 |

### 9.2 后端优化

| 优化项 | 方案 | 效果 |
|--------|------|------|
| 异步请求 | 使用 httpx.AsyncClient | 提高并发能力 |
| 账号轮换 | 多账号随机选择 | 分散请求压力 |
| 请求缓存 | 缓存热门视频数据 | 减少重复请求 |
| 超时控制 | 30秒请求超时 | 防止请求阻塞 |

### 9.3 关键代码优化

```javascript
// 使用 ref 避免闭包问题
const [isPausedRef] = useState({ current: false });

useEffect(() => {
  isPausedRef.current = isPaused;
}, [isPaused]);

// 在异步函数中使用
while (isPausedRef.current) {
  await new Promise(resolve => setTimeout(resolve, 100));
}
```

## 10. 安全考虑

### 10.1 Cookie安全

| 风险 | 措施 |
|------|------|
| Cookie泄露 | 仅在本地存储，不上传服务器 |
| Cookie过期 | 后端自动标记过期账号并切换 |
| XSS攻击 | 前端使用React自动转义 |

### 10.2 请求安全

| 风险 | 措施 |
|------|------|
| CSRF攻击 | 后端添加CORS限制 |
| 请求伪造 | 后端验证Cookie有效性 |
| 频率限制 | 前端添加请求间隔 |

### 10.3 数据安全

| 风险 | 措施 |
|------|------|
| 数据泄露 | 导出文件保存在用户本地 |
| 文件系统 | 使用浏览器原生API，需用户授权 |
