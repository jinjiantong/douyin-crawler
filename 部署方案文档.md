## 3. 后端部署

### 3.1 使用 Gunicorn + Uvicorn Workers

#### 安装 Gunicorn

```bash
source venv/bin/activate
pip install gunicorn
```

#### 创建启动脚本

```bash
# start_backend.sh
#!/bin/bash

# 配置
APP_NAME="douyin-crawler"
APP_DIR="/var/www/douyin-crawler/Crawler-main"
VENV_DIR="$APP_DIR/venv"
PORT=8080
WORKERS=4
TIMEOUT=120

# 日志配置
LOG_DIR="/var/log/douyin-crawler"
mkdir -p $LOG_DIR

# 激活虚拟环境并启动
cd $APP_DIR
source $VENV_DIR/bin/activate

exec gunicorn main:app \
    --workers $WORKERS \
    --worker-class uvicorn.workers.UvicornWorker \
    --bind 127.0.0.1:$PORT \
    --timeout $TIMEOUT \
    --access-logfile $LOG_DIR/access.log \
    --error-logfile $LOG_DIR/error.log \
    --log-file $LOG_DIR/gunicorn.log \
    --daemon

echo "Backend started on port $PORT"
```

#### 使用 PM2 管理进程（推荐）

```bash
# 创建 PM2 配置文件 ecosystem.config.js
module.exports = {
  apps: [{
    name: 'douyin-crawler-backend',
    script: 'venv/bin/python',
    args: 'main.py',
    cwd: '/var/www/douyin-crawler/Crawler-main',
    interpreter: 'none',
    instances: 2,
    autorestart: true,
    watch: false,
    max_memory_restart: '500M',
    env: {
      NODE_ENV: 'production'
    },
    error_file: '/var/log/douyin-crawler/pm2-error.log',
    out_file: '/var/log/douyin-crawler/pm2-out.log',
    log_date_format: 'YYYY-MM-DD HH:mm:ss Z',
    merge_logs: true
  }]
};
```

```bash
# 启动服务
pm2 start ecosystem.config.js

# 保存 PM2 配置
pm2 save

# 设置开机自启
pm2 startup
```

### 3.2 后端配置

#### main.py 配置

```python
# Crawler-main/main.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from service.douyin.urls import router as douyin_router

app = FastAPI(title="Douyin Crawler API", version="1.0.0")

# CORS 配置（生产环境建议限制来源）
app.add_middleware(
    CORSMiddleware,
    allow_origins=["https://your-domain.com", "http://your-domain.com"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# 注册路由
app.include_router(douyin_router, prefix="/douyin")

@app.get("/")
async def root():
    return {"message": "Douyin Crawler API", "version": "1.0.0"}

@app.get("/health")
async def health_check():
    return {"status": "healthy"}
```

## 5. Nginx 配置

### 5.1 安装 SSL 证书

```bash
# 使用 Let's Encrypt 免费证书
sudo apt install -y certbot python3-certbot-nginx

# 获取证书（需要域名解析已配置）
sudo certbot --nginx -d your-domain.com

# 自动续期测试
sudo certbot renew --dry-run
```

### 5.2 Nginx 配置文件

```nginx
# /etc/nginx/sites-available/douyin-crawler

server {
    listen 80;
    server_name your-domain.com;
    
    # 强制跳转到 HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name your-domain.com;

    # SSL 证书配置
    ssl_certificate /etc/letsencrypt/live/your-domain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/your-domain.com/privkey.pem;
    
    # SSL 安全配置
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers off;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 10m;

    # 安全头
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    # 前端静态文件
    root /var/www/douyin-crawler/douyin-crawler-frontend/dist;
    index index.html;

    # Gzip 压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_proxied any;
    gzip_types text/plain text/css text/xml text/javascript application/javascript application/json application/xml;

    # 前端路由
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # API 反向代理到后端
    location /api/ {
        proxy_pass http://127.0.0.1:8080/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        
        # 超时配置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    # 抖音 API 代理（如果需要）
    location /douyin/ {
        proxy_pass http://127.0.0.1:8080/douyin/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        
        # 大文件上传支持
        client_max_body_size 10M;
    }

    # 日志配置
    access_log /var/log/nginx/douyin-crawler-access.log;
    error_log /var/log/nginx/douyin-crawler-error.log;
}
```

### 5.3 启用站点配置

```bash
# 创建符号链接
sudo ln -s /etc/nginx/sites-available/douyin-crawler /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重载 Nginx
sudo systemctl reload nginx

# 开机自启
sudo systemctl enable nginx
```

## 6. Docker部署（可选）

### 6.1 创建 Dockerfile

#### 后端 Dockerfile

```dockerfile
# Crawler-main/Dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    gcc \
    g++ \
    && rm -rf /var/lib/apt/lists/*

# 复制依赖文件
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 复制应用代码
COPY . .

# 暴露端口
EXPOSE 8080

# 启动命令
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8080"]
```

#### 后端 requirements.txt

```
fastapi>=0.109.0
uvicorn>=0.27.0
httpx>=0.26.0
pyexecjs>=1.8.0
```

#### 前端 Dockerfile

```dockerfile
# douyin-crawler-frontend/Dockerfile
FROM node:18-alpine as build

WORKDIR /app

COPY package*.json ./
RUN npm ci

COPY . .
ARG VITE_API_BASE=https://api.your-domain.com
ENV VITE_API_BASE=$VITE_API_BASE

RUN npm run build

# 生产镜像
FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

#### 前端 nginx.conf

```nginx
server {
    listen 80;
    server_name localhost;
    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://backend:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 6.2 Docker Compose 配置

```yaml
# docker-compose.yml
version: '3.8'

services:
  backend:
    build: ./Crawler-main
    container_name: douyin-backend
    ports:
      - "8080:8080"
    environment:
      - PYTHONUNBUFFERED=1
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8080/health"]
      interval: 30s
      timeout: 10s
      retries: 3

  frontend:
    build: ./douyin-crawler-frontend
    container_name: douyin-frontend
    ports:
      - "80:80"
    depends_on:
      - backend
    restart: unless-stopped
    environment:
      - VITE_API_BASE=http://localhost:8080

  nginx:
    image: nginx:alpine
    container_name: douyin-nginx
    ports:
      - "443:443"
    volumes:
      - ./nginx/nginx.conf:/etc/nginx/conf.d/default.conf
      - ./ssl:/etc/nginx/ssl
    depends_on:
      - backend
      - frontend
    restart: unless-stopped
```

## 8. HTTPS 配置

#### Let's Encrypt（推荐）

```bash
# 安装 Certbot
sudo apt install -y certbot python3-certbot-nginx

# 申请证书
sudo certbot --nginx -d your-domain.com -d www.your-domain.com

# 自动续期（Certbot 自动配置）
```

#### 阿里云 SSL 证书

```bash
# 1. 在阿里云 SSL 证书控制台申请证书
# 2. 下载证书（Nginx 格式）
# 3. 上传到服务器
scp cert.zip root@your-server:/etc/nginx/ssl/

# 4. 解压并配置
cd /etc/nginx/ssl
unzip cert.zip
```

### 8.2 证书配置

```nginx
# Nginx SSL 配置
server {
    listen 443 ssl http2;
    server_name your-domain.com;

    ssl_certificate /etc/nginx/ssl/your-domain.com.pem;
    ssl_certificate_key /etc/nginx/ssl/your-domain.com.key;
    
    # 其他配置...
}
```

### 8.3 证书自动续期

```bash

# 添加到 crontab
sudo crontab -e
```

## 10. 常见问题

### 10.1 后端启动失败

**问题**：端口被占用

```bash
# 查看端口占用
lsof -i :8080

# 或
netstat -tlnp | grep 8080

# 杀死占用进程
kill -9 <PID>
```

**问题**：依赖安装失败

```bash
# 使用国内镜像
pip install -i https://mirrors.aliyun.com/pypi/simple/ -r requirements.txt
```

### 10.2 前端构建失败

**问题**：Node 版本不兼容

```bash
# 检查 Node 版本
node -v

# 使用 n 升级 Node
sudo npm install -g n
sudo n 18
```

**问题**：内存不足

```bash
# 增加 Node 内存限制
export NODE_OPTIONS="--max-old-space-size=4096"
npm run build
```

### 10.3 Nginx 502 错误

**问题**：后端服务未启动

```bash
# 启动后端
pm2 restart douyin-crawler-backend

# 检查后端日志
pm2 logs douyin-crawler-backend
```

**问题**：权限问题

```bash
# 检查文件权限
chown -R www-data:www-data /var/www/douyin-crawler
```

### 10.4 HTTPS 证书问题

**问题**：Let's Encrypt 证书申请失败

```bash
# 检查域名解析
nslookup your-domain.com

# 检查防火墙
sudo ufw status

# 手动验证
sudo certbot certonly --webroot -w /var/www/douyin-crawler -d your-domain.com
```

### 10.5 性能问题

**问题**：响应缓慢

```bash
# 检查系统资源
top
htop

# 检查 PM2 内存使用
pm2 list

# 增加 worker 数量
```
