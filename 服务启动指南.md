# 抖音评论采集工具 - 服务启动指南

## 环境要求

- Python 3.11（安装路径：`C:\Program Files\Python311`）
- Node.js（用于 JavaScript 运行环境）
- Windows 操作系统

## 目录结构

```
评论分析/
├── Crawler-main/          # 后端服务
│   ├── venv/              # Python 虚拟环境（需手动创建）
│   ├── main.py            # 后端入口
│   ├── requirements-windows.txt  # Windows 依赖
│   ├── start_server.bat   # 启动后端脚本
│   └── setup_venv.bat     # 创建虚拟环境脚本
│
└── douyin-crawler-frontend/  # 前端服务
    ├── start_frontend.bat   # 启动前端脚本
    └── package.json
```

## 启动步骤

### 方法一：使用脚本一键启动

#### 1. 创建虚拟环境并安装依赖（首次运行）

双击运行 `Crawler-main/setup_venv.bat`

此脚本会：
- 使用 Node.js 创建 Python 虚拟环境
- 安装所有 Python 依赖包

#### 2. 启动后端服务

双击运行 `Crawler-main/start_server.bat`

- 后端服务地址：http://localhost:8080
- 按 `Ctrl+C` 可停止服务

#### 3. 启动前端服务

双击运行 `douyin-crawler-frontend/start_frontend.bat`

- 前端服务地址：http://localhost:3000

## 常见问题

### Q: 提示 "python 不是内部或外部命令"

**解决方案**：使用完整路径 `C:\Program Files\Python311\python.exe`

### Q: pip 安装依赖失败（中文路径问题）

**解决方案**：使用 `requirements-windows.txt`，避免使用包含中文字符的路径作为工作目录

### Q: 添加账号时提示 "添加账号失败，检查后端服务是否运行"

**原因**：后端缺少 CORS 配置

**解决方案**：
1. 确保 `main.py` 包含以下代码：

```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

2. 重启后端服务

### Q: execjs 提示找不到 JavaScript 运行时

**解决方案**：设置 NODE_PATH 环境变量

```batch
set NODE_PATH=C:\Program Files\Adobe\Adobe Creative Cloud Experience\libs
```

## 注意事项

1. **启动顺序**：先启动后端，再启动前端
2. **环境变量**：NODE_PATH 用于 PyExecJS 找 JavaScript 运行时
3. **依赖版本**：Windows 环境使用 `requirements-windows.txt`，避免 uvloop（不支持 Windows）
